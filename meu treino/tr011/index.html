A diferença entre os atributos id e class

id

  • o atributo id especifica uma id exclusiva para uma tag
  • o valor do atributo id não pode se repetir na mesma página
  • o valor não pode ter espaços
  • o valor do atributo id diferencia maiúsculas de minúsculas
  • o atributo id serve para:
    • criar uma área que pode ser acessada através de um hiperlink
    • estilizar com CSS
    • JavaScript

Class

  • o atributo class especifica uma ou mais classes para uma tag
  • várias tags podem compartilhar a mesma classe
    (ao contrário da id, que só pode haver 1 com aquele nome em cada página)
  • o atributo class pode ser usado em qualquer tag
  • o valor do atributo class diferencia maiúsculas de minúsculas
  • uma tag pode ter mais de uma classe (<h2 class="city main">)
    city e main são 2 classes e cada uma terá seu seletor (.city e .main)
  • seletores que se referem a classes começam com um ponto
  • tags diferentes podem possuir a mesma classe e compartilharão o mesmo estilo
    (<h2 class="city"> e <p class="city">)
