Linguagens usadas para a criação de websites

HTML
HTML é uma linguagem de marcação utilizada para criar sites. Ela descreve o significado do conteúdo através de marcações. Atualmente está na sua 5a versão.
CSS
CSS é uma linguagem de marcação utilizada para dar forma/estilo ao conteúdo de um site.
